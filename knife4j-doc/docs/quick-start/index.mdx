:::tip 提示信息

如果开发者使用的是Knife4j 2.x版本，并且Spring Boot版本高于2.4,那么需要在Spring Boot的yml文件中做如下配置：

```yml
spring:
    mvc:
        pathmatch:
            # 配置策略
            matching-strategy: ant-path-matcher
```

:::
